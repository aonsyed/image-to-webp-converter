== Description ==

This plugin converts images to WebP or AVIF format upon upload or via bulk conversion. It includes a WP CLI command and a user interface for bulk operations in the WordPress admin area.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/image-to-webp-converter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= How do I convert images to WebP or AVIF format? =

You can convert images to WebP or AVIF format automatically upon upload, or use the WP CLI command or admin interface for bulk conversion.

= What are the requirements for this plugin? =

The Imagick extension must be installed and enabled on your server.
